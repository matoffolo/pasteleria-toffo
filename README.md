## Clonar, instalar e iniciar el proyecto.
##### `Version de Node: v16.14.1`

```
clon de git https://github.com/matoffolo/pasteleria-toffo.git
cd pasteleria-toffo
npm install
```

Ejecute la aplicación
```
npm start
```
